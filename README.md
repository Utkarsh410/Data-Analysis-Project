# 📊 Data Analysis & Business Intelligence

## 🚀 Overview
This repository contains the **Analytics Layer** of the Data Warehouse project. It focuses on extracting actionable insights from the Gold Layer through Exploratory Data Analysis (EDA) and business-driven SQL reporting.

## 🔍 Key Analysis Areas
- **Exploratory Data Analysis (EDA)**: Profiling data distributions, identifying outliers, and validating data integrity across the Silver layer.
- **Customer Insights**: Segmentation of CRM data to identify high-value customer demographics.
- **Sales Performance**: Analyzing revenue trends, product growth, and seasonal performance using Window Functions.

## 🛠️ Analytics Stack
- **Languages**: T-SQL (Advanced Aggregations, CTEs, Window Functions)
- **Domain**: Business Intelligence, Sales Analytics, Customer Segmentation

## 📂 Analytics Structure
- `/scripts/gold/`: SQL views and scripts for business KPIs.
- `/docs/data_catalog.md`: Documentation of business metrics and field definitions.
